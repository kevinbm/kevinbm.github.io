Research Interests

My current research interests are:
  • Graphical Models, Deep Learning, Neural Networks.
  • Support Vector Machines, Gaussian Processes, i.e. Kernel methods.
  • Scalable Machine Learning. I like Machine Learning because I enjoy doing all the math behind it. Furthermore, what makes it more interesting for me is that almost the same algorithms can be applied to problems from different fields. However, nowadays one has to deal with huge amounts of data and I'm interested in developing methods that can process huge quantity of data as fast as possible without losing generalization.
  • All sort of applications. For instance in: natural language, computer vision, speech processing, biology, spacecraft, etc.
